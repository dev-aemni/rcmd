# RCMD - Remote Terminal Access

Secure remote terminal access with relay server support for global connectivity.

## Features

- 🔒 Password-protected sessions
- 🌐 Global access via relay server (no port forwarding needed)
- 📊 Detailed system metadata collection
- 🐧 WSL detection and information
- 🔄 Real-time command execution
- 👥 Multiple client support
- 🛡️ Encrypted WebSocket communication

## Installation

```bash
npm install -g @dev-aemni/rcmd
